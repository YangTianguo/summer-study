## 四、分科学习策略

### 🔢 数学（每日约3小时）

**核心问题：** 七年级基础计算不过关，概念理解模糊

**学习内容安排：**

| 阶段 | 时间 | 内容 |
|------|------|------|
| 第一阶段 | 第1-2周 | **有理数运算**：正负数加减乘除、乘方、科学记数法（七上第一章） |
| 第二阶段 | 第3-4周 | **整式加减 + 一元一次方程**（七上第二、三章） |
| 第三阶段 | 第5-6周 | **二元一次方程组 + 不等式**（七下） |
| 第四阶段 | 第7-8周 | **几何初步**：线段、角、平行线（七上+七下） |

**学习方法：**
1. 每天先做 **20道计算题**（限时20分钟），这是基本功中的基本功
2. 看课本例题 → 盖住答案自己写 → 对照答案 → 找出错误原因
3. **错题本** 是数学提分最关键的工具：每道错题写清楚【错因+正确解法+同类题1道】
4. 每周日做一套综合测试，检验本周学习效果

**推荐教辅：** 《五年中考三年模拟》七年级分册（只做基础题和中档题）

---

### 🔤 英语（每日约2.5小时）

**核心问题：** 词汇量极度匮乏（估计不足200词），语法概念为零

**学习内容安排：**

| 阶段 | 时间 | 内容 |
|------|------|------|
| 全程 | 每天 | **单词**：每天背15个新词 + 复习之前所有单词（滚动记忆） |
| 第1-3周 | 前3周 | **音标**：48个国际音标全部掌握，做到见词能读 |
| 第1-4周 | 前4周 | **七上单词+课文**：全部过一遍，重点课文要背诵 |
| 第5-6周 | 中2周 | **七下单词+课文** + **一般现在时/现在进行时** |
| 第7-8周 | 后2周 | **一般过去时** + **There be句型** + 初二上册前2单元预习 |

**学习方法：**
1. **音标是第一关**：B站搜索"英语音标教学"，每天学4-5个，一周内过完
2. **单词记忆法**：用"艾宾浩斯记忆法"——新词当天复习3次，第1/2/4/7/15天各复习一次
3. **语法**：不要死记规则，每学一个语法就造5个句子
4. **听力**：每天午餐/晚餐时播放英语课文录音（"磨耳朵"），不求听懂每个词
5. **朗读**：每天早晨大声朗读20分钟，出声读才能形成语感

**推荐工具：** 百词斩APP（背单词）、人教版七上/七下课本配套录音

---

### 📚 语文（每日1.5小时）

**核心问题：** 基础还行，阅读和作文是短板

**学习内容安排：**

| 模块 | 内容 | 频次 |
|------|------|------|
| 古诗文 | 七上+七下所有古诗文背诵默写（逐篇过关） | 每天20分钟 |
| 文言文 | 课内文言文字词解释+翻译（《世说新语》《论语》《狼》《卖油翁》等） | 每周3次 |
| 阅读理解 | 每天做1篇现代文阅读（记叙文为主），做完对答案总结答题模板 | 每天1篇 |
| 写作 | 每周写1篇作文（600字），主题可参考中考常考类型 | 每周1篇 |
| 名著 | 阅读《朝花夕拾》《西游记》（初二上要求） | 每天课外30分钟 |

**阅读答题模板（必背）：**
- 概括题：谁 + 做了什么 + 结果
- 赏析题：修辞手法 + 内容 + 表达效果 + 作者情感
- 含义题：表层含义 + 深层含义（象征/比喻）

### 🌍 历史 / 地理 / 政治（每日轮换，每科每周2次）

这三科的特点是 **背了就有分**，暑假集中突破性价比极高。

#### 🏛️ 历史
- **方法**：画时间轴。把七上（中国古代史：夏商周→秦汉→三国两晋南北朝→隋唐）和七下（宋元明清）的朝代顺序、重大事件、重要人物串成一条线
- **工具**：课本目录就是最好的提纲，先把目录背熟
- **每日任务**：每天背1课的"知识点清单"，闭眼能复述才算过

#### 🗺️ 地理
- **方法**：地图是地理的灵魂。准备一张世界地图和一张中国地图贴在墙上
- **重点**：七上（地球与地图、天气与气候）、七下（世界区域地理）
- **每日任务**：每天记1个区域的地理特征（位置、地形、气候、经济），对着空白地图默写

#### ⚖️ 政治
- **方法**：政治答题有固定套路——"是什么 + 为什么 + 怎么做"
- **重点**：七上（成长的节拍、友谊、师长）、七下（青春、情绪、集体、法治）
- **每日任务**：每天背1课核心知识点 + 做2道材料分析题，模仿标准答案的答题格式

- 用"结构 → 功能"的逻辑串联知识点

## 六、给家长的建议

1. **创造学习环境**：固定学习位置，学习时间全家保持安静
2. **远离手机**：学习时段手机交给家长保管，用的时候再拿
3. **多鼓励少批评**：孩子基础薄弱，提升需要时间，进步了就表扬
4. **每日检查签字**：在孩子的当日学习计划表上签字，让他知道家长在关注
5. **关注心理健康**：差距大容易产生畏难和厌学情绪，及时沟通疏导
6. **适当奖励**：每完成一周计划，可安排一次孩子喜欢的活动（看电影、运动等）
